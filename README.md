# color-classification

Enrich Acelab **company colors** (manufacturers' proprietary color names like
*"Fall River Glaze"*) into standard, searchable color attributes, then make them
searchable by natural-language description (*"warm terracotta matte"*).

Built **standalone against a mocked data layer** first, to be integrated into
Acelab's Directus + Search API + Hatchet stack once proven.
